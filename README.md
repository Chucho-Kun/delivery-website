# Delivery Website | SASS - CSS minificator - WEBP & AVIF image generator
Website made with SASS files and .webp images generate automatically
## How to use
1.- First download the libraries locally
```
npm install
```
2.- Then run the following command to convert images and to observe changes within folders img/ and scss/
```
gulp
```
3.- Finally you can run your preferred server from your Development Environment or use the following command to use a live-server

 ```
npm install live-server --save-dev

live-server
```
